Oscam and cccam installation for linux (ubuntu 11.10 and 12.04 and 12.10)

Written by vaheeD on February 6, 2013
5.00 avg. rating (96% score) - 2 votes

What we need:

  • TV card with chip
  • Phoenix card reader connected to USB port
  • OScam
  • CCCam
  • TVHeadend
  • XBMC
  • UBUNTU

I do not recommend update card with EMM by a phoenix reader, because it can block your chip card. Better is to use original DVB slot for that purpose.

OSCam installation for me stable version is 1.10.

You can check other version there http://www.streamboard.tv/.

sudo apt-get install subversion cmake build-essential libusb-1.0-0-dev libpcsclite-dev
svn checkout http://www.streamboard.tv/svn/oscam/tags/1.10 oscam-1.10
cd oscam-1.10/ ; mkdir build_dir ; cd build_dir ; cmake .. -DWEBIF=1

You should see comething like that

--   operating system: Linux
--   system type: x86_64-pc-linux
--   system: linux
--   cpu: x86_64
--   hw: pc
--   use built-in crypt functions
--   use system pthread functions
--
-- Configuring done
-- Generating done
-- Build files have been written to: /some_path/oscam-1.10/build_dir

Then execute:

#for 4 cores and more use parameter j and core number
make -j4

sudo make install

Then you have to prepare configuration and a log dir. Create folder for that:

sudo mkdir /var/log/oscam
sudo mkdir /etc/oscam/

then you have to create files in /etc/oscam/ examples:
oscam.conf

[global]
usrfile                       = /var/log/oscam/oscamuser.log
logfile                       = /var/log/oscam/oscam.log
cwlogdir                      = /var/log/oscam/cw
nice                          = -1

[monitor]
port                          = 988
aulow                         = 120
hideclient_to                 = 15
monlevel                      = 1
appendchaninfo                = 0

#newcamd section
[newcamd]
#your port 678 for tvheadend then camid and provid
port                          = [email protected]:000068
#newcamd key for DES encryption
key                           = 0102030405060708091011121314

[webif]
#port for frontend in webbrowser
httpport                      = 8989
httprefresh                   = 5
#ip's for web access
httpallowed                   = 255.255.255.255,255.255.255.255,127.0.0.1,192.168.1.0-192.168.1.255,192.168.0.0-192.168.0.255,192.168.10.0-192.168.10.255

[anticasc]
enabled                       = 1
numusers                      = 1
samples                       = 5
penalty                       = 1
aclogfile                     = /var/log/oscam/aclog.log
denysamples                   = 4

TVHEADEND configuration.

Now we will put that oscam to the tvheadend script. Here we go:

create script /etc/oscam/startoscam.sh

sudo touch /etc/oscam/startoscam.sh
sudo chmod +x /etc/oscam/startoscam.sh
sudo vi /etc/oscam/startoscam.sh

and put there:

#!/bin/sh

case "$1" in
  start)
    	echo -n "Stopping oscam..."
	killall oscam > /dev/null
        echo "Starting oscam..."
        /usr/local/bin/oscam -b -c /etc/oscam
        echo " "
  ;;
  stop)
    echo -n "Stopping oscam..."
    killall oscam > /dev/null
echo " "
  ;;
esac
exit 0
}

Then edit /etc/init.d/tvheadend and put that line

	sh /etc/oscam/startoscam.sh start

here:

case "$1" in
  start)
	sh /etc/oscam/startoscam.sh start
	[ "$VERBOSE" != no ] && log_daemon_msg "Starting $DESC" "$NAME"
	do_start
	case "$?" in
		0|1) [ "$VERBOSE" != no ] && log_end_msg 0 ;;
		2) [ "$VERBOSE" != no ] && log_end_msg 1 ;;
	esac
	;;
